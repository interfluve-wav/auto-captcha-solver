# Security Policy

## Supported Versions

Only the latest stable release receives security updates:

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | ✅ (latest: 0.1.4) |
| < 0.1   | ❌                 |

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability in `auto-captcha`, please report it privately:

📧 **Email**: security@interfluve-wav.github.io

**Please do not** open a public GitHub issue for security bugs.

### What to include

- Description of the vulnerability
- Steps to reproduce (PoC helpful)
- Affected version(s)
- Suggested fix (if any)

We aim to respond within 72 hours and issue a patch within 14 days for confirmed issues.

## Acknowledgments

We thank security researchers who responsibly disclose findings. Crediting in release notes is optional — let us know if you'd like to be named.
